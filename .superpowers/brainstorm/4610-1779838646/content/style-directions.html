CSC-CLI 终端风格方向

范围：REPL、消息流、权限弹窗、状态行、工具调用展示。点击一个方向也可以，终端回复更重要。

A. Command Deck / 命令舱
CSC CONTROL /repo/csc sonnet plan
01 agent scanning workspace
02 tool Grep found 14 matches
PERMISSION BashTool bun run typecheck
ctrl+enter accept · esc deny · o inspect

推荐：命令舱 + 轻社区感

最适合高频 CLI。用清晰状态轴、短标签、分层边框和少量品牌橙建立记忆点；欢迎页和成功反馈保留一点人味。

B. Annotated Code / 代码批注
// assistant reasoning
- old guard condition
+ narrowed permission mode
@note Run tests after edit.
review · patch · verify

代码批注流

适合强调 diff 和审阅，但长期对话、权限、后台任务会显得像代码审阅工具，覆盖面偏窄。

C. Community Companion / 社区伙伴
Welcome back, builder.
Clawd is watching the build queue.
✓ typecheck passed
! one permission needs you
warm prompts · friendly empty states

社区伙伴感

记忆点强，但如果作为主轴，容易干扰专业效率；更适合做欢迎页、空状态、低频提示的辅助层。
